Next.js App

This project was created with NextJS, Prisma ORM, MongoDB and Tailwind CSS for styling. 
If you want to run it locally, follow the steps below.

Getting Started
1. Install the dependencies
npm install

2. Start the development server
npm run dev

Once it starts, open your browser and go to:
http://localhost:3000
